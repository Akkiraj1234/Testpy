# Supported Frameworks

Testpy currently supports these frameworks:

| Language | Framework |
| --- | --- |
| Python | pytest |
| JavaScript | Jest |
| C / C++ | Criterion |

## Version compatibility

Framework parsing is version-sensitive, so supported versions should be documented as adapters are added.

## V1 target

- pytest
- Jest
- Criterion

## Future support

- unittest
- Vitest
- GoogleTest
- Catch2
- Rust `cargo test`
- Go `go test`

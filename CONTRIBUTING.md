# Contributing

Thanks for helping improve Testpy.

## Before you open an issue

Please include:
- operating system
- Python version
- Testpy version
- steps to reproduce
- expected behavior
- actual behavior

## Development setup

```bash
git clone https://github.com/Akkiraj1234/Testpy.git
cd Testpy
poetry install
```

## Useful commands

```bash
poetry run pytest
poetry run testpy
poetry build
```

## Pull requests

- keep changes focused
- add tests when logic changes
- update docs when user-facing behavior changes
- keep the code style consistent

## Project philosophy

Testpy should stay:
- simple
- fast
- predictable
- keyboard friendly
